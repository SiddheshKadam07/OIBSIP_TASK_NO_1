# 🎙️ Nova Voice Assistant

## 📌 Project Description

Nova Voice Assistant is a Python application that listens to voice commands and performs tasks such as telling the time, checking the weather, searching Wikipedia, opening websites, sending emails, setting reminders, and more.

This project was developed as part of the **Oasis Infobyte Internship (OIBSIP)**.

---

## ✨ Features

- Voice Recognition
- Text-to-Speech Response
- Tell Time and Date
- Weather Information
- Wikipedia Search
- Open Google and YouTube
- Web Search
- Tell Jokes
- Voice and Keyboard Input
- Simple and Easy-to-Use Interface

---

## 🛠️ Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- Wikipedia
- Requests
- Webbrowser

---

## ▶️ How to Run

1. Download or clone the project.
2. Open the project folder.
3. Install the required libraries:

```bash
pip install SpeechRecognition pyttsx3 wikipedia requests pyaudio
```

4. Run the following command:

```bash
python nova_assistant.py
```
